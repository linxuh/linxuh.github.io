Git命令总结

July 4, 2020

最近入职实习之后发现Git的使用还是需要很熟练，在这篇里面记录一下常用的Git命令，本文档持续更新 git config 在新机器上用git时，要先设置自己的名字和邮箱自报家门，-global表示在所有的本地仓库中生效 git config -global user.name 'yourname' git config -global user.email 'your@email' git config -l //查看所有配置 git init 在项目文件夹中执行 git init 初始化一个版本仓库，会在对应文件夹下出现 .git 文件同时创建工作区 .git 文件记录git项目的各种操作，git init --bare 会创建裸仓库不会创建工作区只有.git git clone git clone git://github.com/gitrepo.git git clone http://github,com/gitrepo.git git clone -b <branchname> <OriginURL> git status git pull git fetch git push git push origin <branchname> -fetch git checkout git checkout -b <branchname> git branch git add git add <filename1> <filename2> .
